## About

**Everforest for Notepad++** brings the beloved Everforest color scheme designed to be warm, soft on the eyes, and easy to work with, to one of the most popular text editors on Windows.

This project aims to cover **all word/style groups** listed in Notepad++'s Style Configurator for each supported language, ensuring consistent and complete highlighting across the board.

The theme is based on Medium/Hard variant of the Dark style, with tiny tweaks for better readability.

## Supported Languages

Currently the following, commonly used languages are completed, with the rest being a work in progress:

- ActionScript
- ADA
- Assembler
- ASP
- ASN.1
- CSS
- C and C++
- HTML
- JSON
- Lua
- XML

Languages not mentioned here may or may not display properly in partial or full extent.

## Installation

1. Download the `Everforest-Dark.xml` file and throw it in `%APPDATA%\Notepad++\themes`
2. Open Notepad++ and go to **Settings > Style Configurator**.
3. Select **Everforest Dark** from the theme dropdown.
4. Click **Save & Close**.
